# segmentation

---

> ### `segmentation_mask.npz`
Compressed numpy array file containing the segmentation mask. This archive holds two arrays: `nuclei` (nuclear segmentation) and `nuclei_exp` (expanded nuclear mask, i.e whole-cell approximation). Label IDs correspond to the `cell_id/label` fields used in [single_cell_data](./single_cell_data.md).
